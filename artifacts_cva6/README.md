# artifacts_cva6

Esta carpeta es exclusivamente para artefactos de tareas del plan SHARC-CVA6.

## Aqui debe ir

- logs
- snapshots
- salidas JSON
- reportes de paridad
- capturas de tests
- trazas E2E
- decisiones de gate por tarea

## Aqui no debe ir

- archivos del flujo principal
- wrapper oficial
- servidor TCP oficial
- launcher oficial
- scripts finales del backend

Esos archivos deben vivir en:

- `SHARCBRIDGE_CVA6/`
